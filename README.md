#### Sobre :
>Repositório referente a conclusão do sexto desafio do Bootcamp GoStack utilizando Node.js junto ao TypeScript dessa vez incluindo o uso de banco de dados com o TypeORM e envio de arquivos com o Multer! A aplicação deve armazenar transações financeiras de entrada e saída e permitir o cadastro e a listagem dessas transações, além de permitir a criação de novos registros no banco de dados a partir do envio de um arquivo csv.

Template da aplicação

#### Como usar:
>Na pasta raiz do projeto utilize o comando `yarn` para instalar todas as dependências do projeto, poderá observar que os resultados sao guiados por testes para executar os testes referentes aos repositórios use o comando `yarn teste __tests__/respositories.spec.js` e o mesmo comando para a parte de likes porem no arquivo  /likes.spec.js
